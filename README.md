# react-native-bubbles-menu

This a bubbles menu

## Installation

```sh
npm install react-native-bubbles-menu
```

Or if you use yarn
```sh
yarn add react-native-bubbles-menu
```

## Usage

```js
import BubbleMenu from 'react-native-bubbles-menu';

// ...

<BubbleMenu items={bubbleItems} />
```
